# Quantum Superposition Entanglement Demo

This example demonstrates fundamental quantum computing concepts: superposition and entanglement. It uses a simple 2-qubit quantum circuit to show how a qubit can exist in multiple states simultaneously (superposition) and how two qubits can become linked such that their fates are intertwined (entanglement). The circuit is simulated locally, illustrating the probabilistic nature of quantum measurements.

## Language

`python`

## How to Run

1. Install Qiskit: `pip install qiskit`
2. Run the script: `python main.py`
